## 🥇 Pool Assistant — Computer Vision XR Project  
### *Best Overall Project Prize Winner — TUD Project Fair 2025*

[![Project Demo](http://img.youtube.com/vi/xSxluX3aaK4/hqdefault.jpg)](https://www.youtube.com/watch?v=xSxluX3aaK4)

An XR pool assistant built for the Meta Quest 3 that uses real-time computer vision and spatial interaction to predict and visualize pool shots.

- Python  
- GDScript  
- WebSockets  
- Computer Vision  
- Meta Quest 3
